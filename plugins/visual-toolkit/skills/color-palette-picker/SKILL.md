# 색상 팔레트 추천기 (Color Palette Picker)

결과물의 목적과 원하는 분위기를 입력받아 색상 이론에 근거한
주색·보조색·강조색 조합을 헥스코드와 함께 추천하고, 접근성까지 체크한다.

## 동작 순서

### 1단계: 입력 파악

사용자 메시지에서 아래 4가지를 추출한다. 명시되지 않은 항목은 합리적으로
가정하되, **결과물 유형**이 불명확하면 그 한 가지만 되묻는다.

- **결과물 유형** (필수): 브랜드 아이덴티티 / 발표 슬라이드 / 웹사이트·앱 UI / 인포그래픽·포스터 / SNS 콘텐츠 / 인쇄물
- **원하는 분위기**: 신뢰·안정 / 활기·발랄 / 고급·세련 / 따뜻·친근 / 차분·미니멀 / 강렬·임팩트
- **사용 맥락**: 디지털 화면 / 인쇄 / 둘 다
- **기존 브랜드 색상**: 있음(어떤 색?) / 없음

### 2단계: 분위기 → 색상 계열 매핑

"원하는 분위기"를 기준으로 색상 계열을 좁힌다. 이게 가장 강한 신호다.

| 분위기 | 1순위 색상 계열 | 보조 계열 | 피할 것 |
|--------|----------------|-----------|---------|
| **신뢰·안정** | 파랑 계열 (#1E3A8A~#3B82F6) | 짙은 회색, 네이비 | 채도 높은 원색 |
| **활기·발랄** | 오렌지·노랑 계열 | 밝은 초록, 산호 | 무채색 주색 |
| **고급·세련** | 검정·짙은 감청·버건디 | 골드, 실버, 오프화이트 | 파스텔, 형광 |
| **따뜻·친근** | 베이지·테라코타·코랄 | 연한 노랑, 연한 초록 | 차가운 파랑, 형광 |
| **차분·미니멀** | 무채색(화이트·라이트그레이) | 연한 파랑, 세이지 그린 | 채도 높은 포인트 과다 |
| **강렬·임팩트** | 빨강·보라·형광 계열 | 검정 배경 강조 | 파스텔, 옅은 색 |

### 3단계: 배색 방식 결정

색상 계열이 정해지면 아래 배색 방식 중 결과물에 맞는 걸 고른다.

| 배색 방식 | 특징 | 적합한 결과물 | 주의 |
|-----------|------|--------------|------|
| **톤온톤(Tone-on-tone)** | 같은 색상, 명도만 달리 | 미니멀 브랜드, 고급 UI | 단조로울 수 있음 |
| **유사색(Analogous)** | 색상환에서 이웃한 색 | 인포그래픽, 발표 | 강조색이 묻힘 |
| **보색(Complementary)** | 색상환 반대편 조합 | SNS, 포스터, 버튼 강조 | 채도 조절 필수 |
| **삼각 배색(Triadic)** | 색상환에서 120° 간격 | 발랄한 브랜드, 어린이용 | 복잡해 보일 수 있음 |
| **분리 보색(Split-complementary)** | 보색의 양옆 두 색 | 균형과 대비 모두 필요할 때 | 보색보다 부드러움 |
| **무채색+포인트 1개** | 흑백 + 단일 강조색 | 임원 발표, 테크 브랜드 | 개성 약할 수 있음 |

**기존 브랜드 색상이 있을 때**: 그 색을 주색으로 고정하고, 위 방식으로 보조·강조색만 도출한다.

### 4단계: WCAG 명도 대비 체크

추천 팔레트에서 **텍스트·배경 조합**에 대해 한 줄 판정을 내린다.

- **AAA (7:1 이상)**: 본문 텍스트에 최적
- **AA (4.5:1 이상)**: 일반 텍스트 통과, 대부분의 용도에 충분
- **AA Large (3:1 이상)**: 18pt 이상 제목·아이콘에만 허용
- **미달**: 장식용으로만 사용, 텍스트 올리지 말 것

흰 배경(#FFFFFF) 또는 검정 배경(#000000) 기준으로 주색의 대비비를 간략히 계산해 표기한다.

### 5단계: 출력

아래 형식으로 답한다. 헥스코드는 반드시 포함한다.

```
🎨 팔레트: [배색 방식 이름] — [분위기 한 단어]

🟦 주색 (Primary)    #XXXXXX  [색 이름]
🟩 보조색 (Secondary) #XXXXXX  [색 이름]
🟧 강조색 (Accent)   #XXXXXX  [색 이름]
⬜ 배경 (Background) #XXXXXX
⬛ 텍스트 (Text)     #XXXXXX

근거: [왜 이 배색 방식인지, 결과물·분위기와 어떻게 맞는지 1~2문장]

♿ 접근성: 주색 위 흰 텍스트 대비비 X.X:1 → [AA 통과 / AA Large만 통과 / 미달]
           텍스트색 위 배경색 대비비 X.X:1 → [등급]

🔄 대안 팔레트:
- [대안1 배색 방식]: 주색 #XXXXXX — [언제 이게 더 나은지]
- [대안2 배색 방식]: 주색 #XXXXXX — [언제 이게 더 나은지]

⚠️ 주의: [이 팔레트의 함정 또는 사용 시 주의사항 한 가지]
```

추가 질문이 없으면 바로 팔레트를 내고, 끝에 "이 팔레트로 실제 슬라이드나
UI 레이아웃 배색을 잡아드릴까요?"를 한 번만 제안한다.

## 색상 계열 레퍼런스

판단 시 아래 표를 근거로 삼는다.

| 색상 | 심리적 연상 | 적합 업종·맥락 | 주의 문화 차이 |
|------|------------|--------------|--------------|
| 파랑 | 신뢰, 전문성, 차분함 | 금융·IT·의료·공공기관 | 서구: 신뢰 / 동아시아: 일부 슬픔 연상 |
| 초록 | 자연, 성장, 안전 | 환경·헬스·식품·금융(성장) | 이슬람권: 신성 / 일부: 돈 |
| 빨강 | 열정, 긴박, 식욕 | 식품·할인·스포츠·경고 | 동아시아: 행운 / 서구: 위험 |
| 노랑 | 낙관, 주목, 에너지 | 어린이·창의·주의 표시 | 채도 높으면 눈에 피로 |
| 보라 | 고급, 창의, 신비 | 뷰티·럭셔리·예술·영성 | 일부 문화: 애도 |
| 오렌지 | 활기, 친근, 식욕 자극 | 식품·소비재·CTA 버튼 | 저렴해 보일 수 있음 |
| 검정 | 세련, 권위, 고급 | 럭셔리·패션·테크 | 서구: 애도 / 동아시아: 다양 |
| 흰색·베이지 | 청결, 미니멀, 여백 | 의료·미니멀 브랜드·발표 배경 | 동아시아 일부: 애도 |

## 예시

**입력**: "스타트업 브랜드 컬러 정하고 싶어. 신뢰감 있으면서 약간 젊은 느낌"

→ 결과물=브랜드, 분위기=신뢰·안정 + 활기 혼합, 맥락=디지털

```
🎨 팔레트: 유사색 (Analogous) — 신뢰+활기

🟦 주색 (Primary)    #2563EB  미드 블루
🟩 보조색 (Secondary) #0EA5E9  스카이 블루
🟧 강조색 (Accent)   #F59E0B  앰버
⬜ 배경 (Background) #F8FAFC
⬛ 텍스트 (Text)     #1E293B

근거: 파랑 계열 유사색으로 신뢰감을 잡고, 앰버 강조색이 CTA·포인트에서
      활기를 더합니다. 스타트업의 "전문적이지만 딱딱하지 않은" 톤에 적합합니다.

♿ 접근성: 주색(#2563EB) 위 흰 텍스트 대비비 4.7:1 → AA 통과
           텍스트(#1E293B) 위 배경(#F8FAFC) 대비비 16.1:1 → AAA 통과

🔄 대안 팔레트:
- 무채색+포인트: 주색 #1E293B — 더 진중하고 테크 회사 느낌이 강할 때
- 보색: 주색 #2563EB + 강조 #EB6325 — SNS용으로 더 강렬한 대비가 필요할 때

⚠️ 주의: 앰버 강조색은 CTA 버튼 1~2곳에만 써야 효과적입니다.
         여러 곳에 쓰면 "세일 느낌"이 나서 브랜드 신뢰도가 떨어집니다.
```

**입력**: "고급스러운 뷰티 브랜드 팔레트, 인쇄물·웹 모두 써야 해"

→ 결과물=브랜드, 분위기=고급·세련, 맥락=인쇄+디지털

```
🎨 팔레트: 톤온톤 (Tone-on-tone) — 고급·세련

🟦 주색 (Primary)    #4A1942  딥 버건디 퍼플
🟩 보조색 (Secondary) #7B3F7A  미디엄 모브
🟧 강조색 (Accent)   #C9A96E  소프트 골드
⬜ 배경 (Background) #FAF7F5  크림 화이트
⬛ 텍스트 (Text)     #1A1A1A

근거: 보라-버건디 톤온톤으로 럭셔리 뷰티의 신비롭고 세련된 이미지를 잡고,
      골드 강조색이 고급 인쇄물에서 메탈릭 효과를 냅니다.

♿ 접근성: 주색(#4A1942) 위 흰 텍스트 대비비 9.2:1 → AAA 통과
           텍스트(#1A1A1A) 위 크림배경(#FAF7F5) 대비비 18.1:1 → AAA 통과

🔄 대안 팔레트:
- 무채색+포인트: 주색 #1A1A1A + 강조 #C9A96E — 더 모던·젠더뉴트럴한 느낌
- 보색: 주색 #4A1942 + 강조 #42A574 — 에메랄드 포인트로 독창성 강조 시

⚠️ 주의: 골드(#C9A96E)는 화면에서 칙칙해 보일 수 있습니다.
         디지털용에선 #D4AF6E로 약간 밝게 조정하고, 인쇄용 별색(Pantone 871C)을
         지정하면 메탈릭 효과를 살릴 수 있습니다.
```
